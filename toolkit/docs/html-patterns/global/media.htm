Media

The media object allows you to have an image/video or other fixed width media to the left or right with some content that describes it in the center.

Note: borders have been added to the examples below to show the differences in spacing.

The idea comes from here: http://www.stubbornella.org/content/2010/06/25/the-media-object-saves-hundreds-of-lines-of-code/

Genius!

Want to take your next step in Database Lifecycle Management?

These one-day workshops will teach you everything you need to get started.
