You can set:

  • .padded
  • .padded-h (horizontal)
  • .padded-v (vertical)
  • .padded-top
  • .padded-bottom
  • .padded-left
  • .padded-right

You can add --loose or --tight to any of these, e.g. .padded--loose, .padded-bottom--loose

Margins

All the same classes exist for margins as well - just replace 'padded' with 'spaced'.
