Scroll to

Scroll to an area of the page that is linked to using an internal anchor. Simply add the "js-scroll-to" class to the internal anchor.

Options

Using HTML data attributes, options can be set for the individual scroll-to components.

OptionDescriptionDefaultExample usage
OffsetAdd an offset for the scrolled to position. For example, if using a sticky element, you may wish to push forward/back where the page scrolls to.0Go to my content
FocusAdd focus to an element after the page has finished scrolling. Could be useful for adding focus to form fields.falseGo to my form and focus on my form field
