Equalise (Equalise heights of elements)

Use HTML classes on elements to equalise their heights.

Using data attributes allows grouping of elements within an equalised container. For example, if you needed to equalise the heights of headings, separately to paragraphs, within a container.

  • Container HTML class: js-equal-heights
  • Item HTML class: js-equal-heights__item
  • Item group HTML data attribute (Optional): data-js-equal-heights-group
