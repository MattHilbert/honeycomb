Tabs

Tabs are used for alternating between views within the same context.

The tabs functionality can utilise ajax to load in content from other pages on the website.

Options

Using HTML data attributes on the tabbed element options can be set to adjust the functionality of the tabbed element.

OptionDescriptionDefaultExample usage
Scroll animationUse an animation effect to scroll to the tab when it's opened.Truedata-tabs-scroll-to="false"
Scroll animation offsetIf using the scroll to animation (see above), then an offset can be set to adjust where the window scrolls to.0data-tabs-scroll-to-offset="10"
PaginationAdd a pagination object to the bottom of each tab, providing links to the next/previous tabs.Falsedata-tabs-pagination="true"
Reload ajax requestsWhen ajaxing content in, should the content be reloaded each time the tab is opened.Falsedata-tabs-reload-ajax="true"
