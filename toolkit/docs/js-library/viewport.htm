In/Out Viewport

This functionality add a class when a selected element scrolls in and out of the viewport.

An example where this could be used is for starting animations on elements when they become in the viewport, rather than them happening where the user cannot see them.

This functionality adds a class of "vp-in" to the element when it's in the viewport and "vp-out" when it's out of the viewport.
