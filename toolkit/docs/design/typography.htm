Typography

The font face stack is "Roboto", "Helvetica Neue", Arial, Sans-serif

The base font size for the website is left at 100% (which is 16px by default in most browsers).

All font-sizes are calculated as REMs, with a pixel fallback value.

The basic set of styles are based on a typographic scale of 11, 14, 16, 18, 24, and 32.

All components align to an 8 pixel square baseline grid and type aligns to a 4px grid, within reason.
