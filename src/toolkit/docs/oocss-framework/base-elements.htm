Base elements

Typographic scale and baseline

The base font size for the website is left at 100% (which is 16px by default in most browsers).

All font-sizes are calculated as REMs, with a pixel fallback value.

The basic set of styles are based on a typographic scale of 11, 14, 16, 18, 24, and 32.

All components align to an 8 pixel square baseline grid and type aligns to a 4px grid, within reason.

Image Sizes

All images should be produced as SVG where possible.

Pixel dimensions for key images sizes:

  • Product logos, in "secondary-nav" component - 36px high
  • Document covers (whitepapers, PDFs, case studies, etc.) - 180px wide
  • Screenshots (XXX-small.png), lightbox image links - 283px x 15px (16:9)
  • Company logos, for logo bands - 240px x 135px (16:9)
  • Hero illustrations on product pages, with tagline / CTA to the left - minimum of 440px wide (can include negative space, as suits illustration placement)
