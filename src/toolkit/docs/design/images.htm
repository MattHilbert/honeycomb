General notes

Use SVG whenever possible, and create a PNG fallback for less able browsers.

Create borders in the image, as the box model rendering will distort the image if a border is added with CSS.

Image sizes

Product logo for the secondary-nav component

Logos should be created as SVG and PNG. The PNG should be 36px high.

For example:

Document covers (whitepapers / case study PDFs)
180px wide
Small screenshot
283px x 158px (16:9)
Company logos
240px x 135px (16:9)
Headshots

400px x 440px

Make the subjects head 220px tall, placed at the top of the image.
